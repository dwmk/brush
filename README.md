# BRUSH

![](/app/opengraph-image.jpg)

Inspired by [PrePreReg](https://preprereg.eniamza.com/), [PreConnect](https://preconnect.app), [B.O.R.A.C.L.E.](https://boracle.app).

Built as a successor to [BRAX](https://braxapp.github.io/).

Special thanks to [Tashfeen Azmaine](https://github.com/eniamza) for his public JSON CDN source from [Connect](https://connect.bracu.ac.bd), 

Built using [v0.app](https://v0.app), [Gemini](https://gemini.google.com) and my own past knowledge and experiences. [[Learn more]](https://dewanmukto.github.io)

UI design based on **neo-brutalism** as seen implemented by [Gumroad](https://gumroad.com), and on various demos on [Dribbble](https://dribbble.com/tags/neo-brutalism) and [Figma](https://www.figma.com/community/file/1613945708230609539/neo-brutalism-ui-library).

UX design based on my own opinions and personal needs, now available publicly for everyone.

Color palette chosen to reflect vibrant highlighter colors with the perfect combinations based on color theory.

Promotional materials (e.g. banners) and everything else by me.
